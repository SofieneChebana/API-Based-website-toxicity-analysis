# API-Based-website-toxicity-analysis

Projet de Master Informatique à l'université de Caen Normandie pour l'année 2024-2025.

"API Based Website Toxicity analytics".

<h2>Instructions:</h2>

1) You will need to install all the dependencies by typing at the root of the project:
 ```
 "pip install -r requirements.txt"
```
2) To open the main program that will let you visualize the classification of the pages from 'projet.db', go to src/ and type :
  ```
  "python3 main.py"
  ```
3) To test the toxicity of an url using the model, go to src/ and type :
  ```
    "python3 urlTester.py".
  ```
The result will be registered in the 'projet.db' database.

4) To begin a new training for the model, go to src/ and type :
  ```
   "python3 model.py"
  ```
